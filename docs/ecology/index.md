# 生态

大部分基础较为常用的功能内置于`usepy`中，但是有一些功能并不是所有人都需要，所以我们将这些功能放在了扩展中，这样可以保证`usepy`更加轻量化。

## 仓库列表

- [use-rabbitmq](https://github.com/use-py/use-rabbitmq)
- [use-nacos](https://github.com/use-py/use-nacos)
- [use-redis](https://github.com/use-py/use-redis)
- [use-logger](https://github.com/use-py/use-logger)
- [use-notify](https://github.com/use-py/use-notify)
